# Investor
Investor Relations (IR) spec for Ruff Management Inc.

## Required Statements
* Finance
  - Income Statement
  - Cash Flow Statement
  - Balance Sheet
  - Note to Financial Statements
  - Statement of Change in Equity
* Management
  - Managed Companies
  - Active Projects/Campaigns
  - Property/Asset Inventory
* PROSPECTUS
